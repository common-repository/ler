== Description ==

 A professional plugin for adding LER on your blog.

 Un plugin profesional pentru adaugarea de LER pe blog.
 
 
== Installation ==
 
1. Download the plugin, extract and upload ler.php in /wp-content/plugins
2. Activate plugins from Admin Panel -> Plugins -> LER
